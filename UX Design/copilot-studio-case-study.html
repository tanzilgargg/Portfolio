Copilot Studio AI Chat Assistant

Conversational Agents Challenge - Building an AI-powered assistant to replace static FAQs with intelligent, grounded responses

My Role

UX Researcher & Conversational Agent Designer

Duration

3 Days

Tools Used

Copilot Studio, Microsoft Learn, Knowledge Curation

Project Overview

Built an AI-powered assistant that answers common questions about Microsoft Copilot Studio using curated internal and external documents. The assistant was designed to replace static FAQs and reduce reliance on live support while maintaining accuracy and trust through grounded responses.

This project was part of the Conversational Agents Challenge in the Microsoft Power Up Program, focusing on creating intelligent, reliable AI assistants that enhance user experience without compromising on accuracy.
