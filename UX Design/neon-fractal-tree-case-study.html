Exploring emotional responses to minimalist visuals and recursive growth patterns without traditional UI elements

Project Overview

The Neon Fractal Tree is a generative art project designed to explore how minimalist visuals and recursive growth patterns can emotionally impact viewers without text, UI, or sound. The project tests how users engage with fractal motion and color-driven feedback in an ambient, digital environment.

Problem + Goals

Problem:

Digital experiences often rely on explicit interaction (buttons, menus, feedback loops). But emotional or meditative responses are harder to achieve without direct guidance.

"How can we create digital experiences that evoke emotional responses without traditional interface elements?"

Goals:

  • Create a screen-based experience that evokes emotional response without using traditional interface elements.
  • Observe how viewers interpret, engage, or disconnect from purely visual, recursive interaction.
  • Evaluate how fractal growth, color dynamics, and cursor movement influence perceived calmness, attention, or curiosity.

Research Plan

Methodologies

Exploratory User Testing

6 participants, aged 18–30, observed while interacting with the fractal animation

Think-Aloud Protocol

Participants verbalized thoughts and feelings while watching the animation

A/B Testing

Comparison of color schemes (neon magenta vs. electric blue) to assess emotional impact

Behavioral Observation

Tracking hover/click behavior over a 2-minute interaction window

Questions Asked

  • What words would you use to describe this experience?
  • What did you expect to happen when the tree grew?
  • Did the visuals feel calming, intense, or disorienting?
  • Would you interact with this again? Why or why not?

Quantitative & Qualitative Summary

  • 83% described the tree growth as "calming" or "mesmerizing"
  • 67% showed more interaction with the blue palette version
  • 4 of 6 participants hovered or clicked multiple times to "see what happens"
  • Emotional terms collected: hypnotic, rebirth, peaceful, futuristic

Findings

User Quotes

"It feels like the tree is alive — like breathing in and out."

"I don't know what it's for, but I kept watching anyway."

"This would make a great visualizer for meditation."

Pain Points

  • Some users expected interaction feedback (click-to-grow, audio, etc.)
  • A few expressed confusion about when the animation loops
  • One user thought it was a loading screen at first glance

Emotional Themes

  • Calmness, stillness, and curiosity
  • Users framed the visual as organic, alive, or spiritual
  • The neon palette prompted a sense of future-forward aesthetics

Insights & Opportunities

  • Visual feedback alone can sustain user attention for ~2 minutes
  • Fractal growth mimics organic motion, triggering emotional resonance
  • Color palette plays a huge role in perceived tone — cooler tones = more relaxing
  • Potential use cases: digital art displays, mindfulness apps, idle background interactions

Brief User Flow

User visits → Sees static tree seed → Tree grows recursively → Subtle cursor reaction → Color pulse with hover → Seamless loop

Proposed Solutions

Cursor Trail Effect

Added a gentle cursor trail effect to increase perceived interactivity without requiring explicit actions.

Seamless Loop Timing

Reworked animation timing to fade seamlessly instead of hard resets, creating a more meditative experience.

Color Palette Refinement

Reworked color schemes with reduced brightness for a softer experience based on user feedback.

Reflection + Lessons Learned

This project expanded my understanding of non-verbal UX — experiences that rely on emotion, rhythm, and visual intuition rather than text or interface logic. It reminded me that users bring their own interpretations, and even the simplest interaction (like watching a fractal tree grow) can carry deep meaning.

"The most valuable insight was discovering how much emotional connection can be created without traditional UI elements — sometimes less interaction creates more engagement."

Moving forward, I'm interested in applying similar emotional UX principles to areas like meditation tech, ambient interfaces, and interactive digital art.
